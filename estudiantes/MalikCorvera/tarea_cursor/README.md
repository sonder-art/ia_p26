# Instrucciones

## Instalación
Asegúrate de tener Cursor instalado y funcionando.

Si estás en Windows, verifica que `cursor .` abra desde tu terminal WSL.

## Crear Carpeta

Crea una carpeta llamada `test_ide` en tu escritorio o home.

Ábrela con Cursor.

## Usar Composer (Plan Mode)

1. Presiona `Ctrl + I` (o `Cmd + I`).

2. Escribe este prompt:

   "Crea un archivo llamado 'hola_ide.md'. Dentro, escribe alguna reflexion filosofica sobre la ia o algun tema visto."

3. Dale a "Submit" y observa cómo crea el archivo.

4. Revisa el cambio y dale a "Accept" (o `Ctrl + Enter`).

## Renderizar Markdown

1. Abre el archivo `hola_ide.md` que se creó.

2. Presiona `Ctrl + Shift + V` (o `Cmd + Shift + V`).

3. Deberías ver el documento con formato bonito (títulos grandes, etc).
