# Módulo 3: Configuración del Sistema Operativo

Para el desarrollo moderno de IA, es indispensable trabajar en un entorno basado en Unix. Este módulo te guiará para configurar tu computadora.

## Contenido

1.  **[Instalación de WSL2 (Windows)](./01_wsl_install.md)**
    *   Si tienes Windows, esta es la guía obligatoria.
    *   Incluye instrucciones para Dual Boot (avanzado).
    *   Usuarios de Mac/Linux nativo pueden saltarse la instalación pero leer para entender los comandos básicos.

2.  **[Entornos en la Nube (Browser)](./02_browser_env.md)**
    *   Si no tienes computadora personal o no puedes instalar nada.
    *   GitHub Codespaces, Gitpod (Ona), Google Colab.

---

## Recomendación General

Si eres nuevo en programación:
*   **Windows:** Usa WSL2. Es lo estándar en la industria hoy en día.
*   **Mac:** Ya estás listo.
*   **Linux:** Ya estás listo.
*   **Chromebook/Tablet o no Laptop:** Usa la opción de Nube (Browser).
