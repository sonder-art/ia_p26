# Entornos en la Nube con IA (Cloud IDEs)

Si no puedes instalar Cursor localmente, estas son las mejores opciones que **ya incluyen IA**.

## 1. GitHub Codespaces (+ Copilot)

Codespaces te da un VSCode completo que corre en los servidores de GitHub.
*   **Ventaja:** No necesitas configurar nada en tu máquina.
*   **La IA:** Para tener IA aquí, necesitas activar la extensión de **GitHub Copilot**.
    *   Como estudiantes, pueden aplicar al [GitHub Student Developer Pack](https://education.github.com/pack) para obtener Copilot gratis.
    *   Una vez activado, funciona muy similar al "Chat" y "Tab" de Cursor, aunque la capacidad de editar múltiples archivos (Composer) es más limitada.

> **Prompt para LLM:**
> "Estoy usando GitHub Codespaces. Tengo mi cuenta de estudiante aprobada. ¿Cómo instalo y activo la extensión de GitHub Copilot dentro de mi codespace para que me ayude a programar?"

## 2. Ona (Anteriormente Gitpod)

Ona es una evolución de Gitpod que se centra fuertemente en **Agentes Autónomos**.
*   **Diferencia:** Mientras Codespaces es un "editor en la nube", Ona busca ser un entorno donde agentes de IA trabajan contigo.
*   **Agentes:** Ona integra agentes que pueden planificar, ejecutar y validar tareas (similar al Composer de Cursor pero en la nube).
*   **Acceso:** Tienen planes gratuitos limitados (normalmente 50 horas/mes).
*   **Integración:** Puedes usar Ona desde el navegador (VSCode web) o conectarlo a tu VSCode/Cursor local.

> **Prompt para LLM:**
> "¿Qué es Ona (antes Gitpod)? ¿Cómo puedo abrir un repositorio de GitHub usando Ona y cómo funcionan sus agentes de IA para ayudarme a escribir código?"
